# Dashboard de Vendas

Dashboard de vendas em tempo real para as lojas de suplementos para pets, consolidando todos os canais de venda numa única tela.

## Canais integrados

**Brasil**
- Shopify (cocoandluna.com.br)
- Shopee
- Mercado Livre
- Amazon.com.br

**EUA**
- Shopify (vita-pet-life.myshopify.com)
- Amazon.com

## O que mostra

- Receita, pedidos e ticket médio por canal e período
- Tendência diária de vendas
- Custo de anúncios e ROAS (Meta Ads — Instagram e Facebook)
- Funil de sessões e conversão (Shopify)
- Distribuição geográfica de pedidos por estado (BR e EUA)
- Top produtos e pedidos recentes

## Stack

Node.js + Express no backend, HTML/CSS/JS puro no frontend. Banco de dados PostgreSQL em produção (Railway).
